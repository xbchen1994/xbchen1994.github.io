Naive Beyes Classifier

Naive Bayes is a conditional probability model: given a problem instance to be classified, represented by a vector $A=(A_{1},A_{2},\cdots,A_{n})$ representing some $n$ features (independent variables), it assigns to this instance probabilities $p(B|A_{1},A_{2},\cdots,A_{n})$ for each of possible outcomes or classes $B$

Then $p(B|A_{i})=\frac{p(B)p(a_{i}|B)}{p(a_{i})}$

In plain English, using Bayesian probability terminology, the above equation can be written as

$posterior=\frac{prior \times likelihood}{evidence}$
